# FLIP 动画示例

复刻小红书 "点击某个卡片 → 页面跳转或路由变化 → 弹窗无缝放大到详情页" 效果， 实现路由驱动 + FLIP 动画方案。

## 运行示例

**重要**：必须在项目根目录运行，而不是在 examples 子目录！

```bash
# 在项目根目录 (view-transition/)
cd d:\CM-Workspace\Bobo\Projects\view-transition

# 安装依赖（如果还没安装）
npm install

# 启动开发服务器
npm run dev
```

然后访问 `http://localhost:5173`

## 功能特性

在上一版本基础上，考虑组件的完整性和健壮性，以及代码上规范等，优化项如下：
1. 规范组件命名，使其符合语义；
2. 更完善的应用场景，不通过点击方式，记录图片位置；
3. 兜底处理，当元素不存在时，不显示动画，直接显示当前元素；
